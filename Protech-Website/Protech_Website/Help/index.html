Introduction

 

The FitPlain- one page Creative portfolio Template is a simple, contrast, stylish and professional look. It suitable for Landing page, visual portfolios, wedesign company, fashion, personal portfolio and business type web site, It's a responsive template so it looks perfect in desktop, ipad and all mobile devices.

Currently the template have 7 modules.

They are
  •   1.  Home page
  •   2.  Product
  •   3.  Team
  •   4.  service
  •   5.  Portfolio - masonry filterable gallery and Video page
  •   6.  Blog page
  •   7.  Contact us page

Menu

The menu is placed inside the div with class name of nav.

You can add the menu using below code 

    <li><a href="#!home" class="first active"> <i class="icon-globe"></i> HOME</a></li>

In each section you must set the page name(id) in data-id attribute, this data-id  is used to link on site menu. see the below example. 

    <div class="contentWrapper" data-id="!home" ></div>

The data-id (!home) is assign in href (#!home) value in menu tag.

If the page have different section(like product page) just use the data-continue-page attribute to define the main page data-id
Example: data-continue-page="!product"

The main use of this attribute (data-continue-page) is to identify the current menu that belong to the page, it will help to highlight the current menu. See the below code 

    <div class="contentWrapper" data-continue-page="!product" data-id="!home" ></div>

You can change the menu icons on <i> tag. This template support font awesome icons. Click here to see the complete list of availiable icons

Parallax

The main structure of the parallax code is show below
 
<div class="contentWrapper parallax" data-src="images/background/image1.jpg" data-src-small="images/background/image1_s.jpg" >
	<div class="parallaxPattern"></div>
	<div class="container" >
		<!-- You can place any content here -->
	</div>
</div>

On the above parallax class div you can see the two attribute data-src and data-src-small is used to add the parallax image. data-src is used to set large size image (for Desktop) and data-src-small is used to set small size image (for Mobile devices) so that Mobile devices will load low resolution version of images to gain performance as well as saves download time.

The property data-src is used to set image path for desktops and data-src_small  for mobile devices.

/* Initialize  - Masonry isotope Portfolio */

  try {
		  $(function(){
			
			var $container = $('.masonry_container');
			var $optionSets = $('#options .option-set'),
				$optionLinks = $optionSets.find('a'),
				 itemSelector = '.item';
	  
			$optionLinks.click(function(){
			  var $this = $(this);
			  
			  // don't proceed if already selected
			  if ( $this.hasClass('selected') ) {
				return false;
			  }
			  var $optionSet = $this.parents('.option-set');
			  $optionSet.find('.selected').removeClass('selected');
			  $this.addClass('selected');
		
			  // make option object dynamically, i.e. { filter: '.my-filter-class' }
				 var options = {},
					key = $optionSet.attr('data-option-key'),
					value = $this.attr('data-option-value');
				// parse 'false' as false boolean
				value = value === 'false' ? false : value;
				options[ key ] = value;
				
				$(itemSelector).each(function(){
					var clss = String(options[ key ]).split(".");
					var cs_ = clss.length > 0 ? clss[1] : itemSelector;
  
					 if( $(this).hasClass(cs_) || clss.length === 1 ){
						  $(this).removeClass("hideItem");
						  $(this).animate({"opacity":1}, 200);
						  $(this).find("a[data-fancy]").attr("data-fancy", 'groupFancy');
						}else{
						  $(this).addClass("hideItem");
						  $(this).animate({"opacity":.15}, 200);
						  $(this).find("a[data-fancy]").attr("data-fancy", '');
						}
					});
				
				if ( key === 'layoutMode' && typeof changeLayoutMode === 'function' ) {
				  // changes in layout modes need extra logic
				  //changeLayoutMode( $this, options )
				} else {
				  // otherwise, apply new options
				  //$container.isotope( options );
				}
			  
			  return false;
			});
			
		  });

$("input#name").val(''); 
$("input#name").focus();  
return false;  
}

// validate Email 
var email = $("input#email").val();  
regEx=/^[A-Za-z0-9._%-]+@[A-Za-z0-9.-]+[A-Za-z]{2,4}$/;

$("input#email").val(''); 
$("input#email").focus();  
return false;  
}  

// validate comment 
var comments = $("textarea#comments").val(); 
if (comments == "" || comments == "Comments..." || comments.length < 2) { 
$("textarea#comments").val(''); 
$("textarea#comments").focus();  
return false;
