HTML Structure - top

Columns

Leopard template has a fixed layout with a variation of column layouts depending on the page. This theme is based on Bootstrap 3.0 Grid System. Bootstrap includes a responsive, mobile first fluid grid system that appropriately scales up to 12 columns as the device or viewport size increases. It includes predefined classes for easy layout options.

The main structure of this site is wrapped in a 'container' div. Inside of that div you will find each major section of the site given an appropriate grid size all equalling 12 columns in length. So you can easily and quickly build any page structure as you want. Example:

If you would like to edit the color, font, or style of any elements in one of these structures, you would do the following:

a {
	color: #fff;
}

If you find that your new style is not overriding, it is most likely because of a specificity problem. Scroll down in your CSS file and make sure that there isn't a similar style that has more weight.

I.E.

	
footer .yourDiv a {
	color: #f3f3f3;
}

Color Changing

To change default color scheme go to ' style.css ' and modify this section: 

/*--------------------------------------------------------------------------*/
/* IMPORT
/*--------------------------------------------------------------------------*/
...

/* COLORS */
@import url("css/colors/color-blue.css"); 

...

You can choose one of this predefined colors:

  • css/colors/color-blue.css
  • css/colors/color-green.css
  • css/colors/color-orange.css
  • css/colors/color-turq.css
  • ... or easily create you own color scheme

Wide / Boxed version

You can also choose beetwen wide ( stretched ) or boxed version. Wide is default, to set the boxed versions go to ' style.css ' and modify this section: 

/*--------------------------------------------------------------------------*/

/*--------------------------------------------------------------------------*/
...

/* SET BOXED */
/* @import url("css/boxed.css"); */ // remove /* */ to set boxed

Flickr Widget

Code responsible for displaying flickr photos: 

<script type="text/javascript" src="http://www.flickr.com/badge_code_v2.gne?count=8&size=s&layout=h&source=user&user=52617155@N08"></script>

For displaying your own Flickr photos go to idGettr.com to get your flickr id and put it instead "52617155@N08"

Twitter Widget

Code responsible for displaying twitter feeds: 

<div class="widget-twitter"></div>

For displaying your own Tweeter Feeds:

  1. You need to have a twitter App for your usage in order to obtain OAuth credentials, see https://dev.twitter.com/apps for help.
  2. After creating your app, configure 'js/twitter/index.php' with your OAuth credentials, and enable caching if you desire. Your cache directory of choice must be writable by PHP.
  3. Go to ' js/custom.js ' and modify this section
/* TWITTER WIDGET */
$(document).ready(function() { 
$('.widget-twitter').tweet({
			modpath: 'js/twitter/',
			username: 'envato',
			join_text: null,
			avatar_size: null,
			count: 3,
			loading_text: 'loading twitter feed...',
			template: "{text}<br>{time}"
	});
});

Contact Form

Setting up the contact form to forward email messages requires that the template is hosted on a server that supports PHP. Please contact your hosting provider if you're unsure of if they do support it.

The only step required to have message directed to your email of course, is to update 'php/send.php' with your own email address as seen below. 

<?php
	if($_POST){
		$to = 'your.email@domain.com'; /*Put Your Email Adress Here*/
		$from_name = $_POST['name'];
		$from_email = $_POST['email'];
		$message = $_POST['message'];
		$subject = "Contact form message: $from_name <$from_email>";
		$header = "From: $from_name <$from_email>";
		mail($to, $subject, $message, $header);
	}
?>
